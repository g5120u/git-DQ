# 角色精靈圖說明 Character Sprites

## 📁 檔案位置

所有角色精靈圖應該放在 `assets/characters/` 資料夾中。

## 🎨 精靈圖格式要求

### 公主角色（princess.png）

**檔案名稱**：`princess.png`

**格式**：
- PNG 格式，透明背景
- 建議尺寸：96x192 像素（32x48 像素 × 3列 × 4行）
- 精靈圖表結構：4行 × 3列
  - 第1行：面向前方（3個動畫幀）
  - 第2行：面向左方（3個動畫幀）
  - 第3行：面向右方（3個動畫幀）
  - 第4行：面向後方（3個動畫幀）

**動畫說明**：
- 每個方向有 3 個動畫幀，用於待機/行走動畫
- 戰鬥系統會自動循環播放這些幀

### 怪物角色

**檔案名稱**：`slime.png` 或 `boss.png`

**格式**：
- PNG 格式，透明背景
- 建議尺寸：96x32 像素（32x32 像素 × 3列 × 1行）
- 精靈圖表結構：1行 × 3列（3個動畫幀）

## 📝 如何添加精靈圖

1. 將精靈圖檔案放在 `assets/characters/` 資料夾中
2. 確保檔案名稱正確：
   - 公主：`princess.png`
   - 史萊姆：`slime.png`
   - Boss：`boss.png`
3. 如果精靈圖不存在，系統會自動使用簡單的形狀作為備用

## 🎮 在戰鬥中的使用

戰鬥系統會自動：
- 根據角色面向選擇對應的精靈圖行
- 循環播放動畫幀
- 在攻擊、受傷、勝利時顯示對應的動畫

## 💡 提示

- 精靈圖應該使用像素藝術風格
- 確保背景透明（PNG alpha 通道）
- 角色應該面向右側（在精靈圖中）
- 動畫應該流暢自然
