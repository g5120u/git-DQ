# 貢獻規範 Contributing to git-DQ

感謝你對 git-DQ 的貢獻興趣！

Thank you for your interest in contributing to git-DQ.

--------------------------------------------------

## 基本規則 Basic Rules

- Fork 本專案並由 main 分支建立新分支  
- 分支命名 Branch naming:
  - feature/xxx
  - fix/xxx

- Commit 訊息格式 Commit message:
  - feat: 說明
  - fix: 說明

--------------------------------------------------

## Pull Request 規則 Pull Requests

- 請清楚說明你的修改內容
- 所有 PR 皆需經過審核，可能被要求修改或拒絕
- 合併後的程式碼將成為 git-DQ 的一部分

By submitting a Pull Request, you agree that your contribution may be used under the project license.
提交 PR 代表你同意你的貢獻內容將依本專案授權使用。
